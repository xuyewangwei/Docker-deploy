```
 systemctl start docker
```

# docker基本命令
```
docker pull	从 Docker Hub 拉取镜像
docker run	运行一个容器
docker ps	查看正在运行的容器
docker ps -a	查看所有容器（包括已停止的）
docker stop	停止正在运行的容器
docker start	启动一个已停止的容器
docker rm	删除容器
docker rmi	删除镜像
docker exec	在运行的容器中执行命令
docker logs	查看容器的日志输出
```
